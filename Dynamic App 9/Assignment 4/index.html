Console Calculator

The calculator functions have been automatically loaded into your browser!

Step 1: Open your browser's Developer Console:

/* Windows / Linux */
Press Ctrl + Shift + J

/* Mac */
Press Cmd + Option + J

Step 2: Test the operations in the console:

> add(10, 5)
// returns 15

> subtract(20, 8)
// returns 12

> multiply(4, 6)
// returns 24

> divide(100, 4)
// returns 25
Look inside the console to see the full list of available commands!
